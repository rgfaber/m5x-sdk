# M5x STAN Library

## Purpose

- Centralized Managment of NATS.Client dependencies
- a wrapper around NATS.Client, STAN.Client and NATS.Client.Extensions.DependencyInjection
- Simplify the use of NATS/STAN for downstream engineering teams.
